## Структура проекта

1. Добавить файл `db.php` в корень проекта с ассоциативным массивом `$db` внутри, содержащим данные для подклчения к БД:

```php
$db = [
  'host' => 'host',
  'user' => 'user',
  'password' => 'password',
  'name' => 'db_name',
];
```
